About

Red Hat Application Migration Toolkit is Red Hat's toolkit that allows developers to quickly decompile, analyze, assess, and migrate legacy applications from various Java EE platforms to JBoss EAP.

Version

Red Hat Application Migration Toolkit Core:
Red Hat Application Migration Toolkit 3.1.0-SNAPSHOT
Red Hat Application Migration Toolkit Web Console:
Red Hat Application Migration Toolkit 3.1.0-SNAPSHOT

The documentation contains information and guides to help you learn about Red Hat Application Migration Toolkit and start exploring its features. From getting started with migration of your first application, to trying out creation of your own rules, it provides what you need to set up and use Red Hat Application Migration Toolkit.
