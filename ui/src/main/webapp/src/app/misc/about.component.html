About

Application Migration Toolkit is Red Hat's toolkit that allows developers to quickly decompile, analyze, assess, and migrate legacy applications from various Java EE platforms to JBoss EAP.

Version

The documentation contains information and guides to help you learn about Red Hat Application Migration Toolkit and start exploring its features. From getting started with migration of your first application, to trying out creation of your own rules, it provides what you need to set up and use Red Hat Application Migration Toolkit.

With the Red Hat Application Migration Toolkit command line interface (CLI), you can create migration reports from the terminal. To get started using the CLI, visit Red Hat Application Migration Toolkit Command Line Interface Console documentation.
